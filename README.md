1만 시간의 법칙 웹 프로젝트

10,000 Hour Rule – Web Application

1. 프로젝트 개요

본 프로젝트는 ‘1만 시간의 법칙(The 10,000 Hour Rule)’을 주제로 한 학습·자기계발 동기부여용 웹 페이지입니다.
사용자가 목표를 설정하고 시간을 인식할 수 있도록 시각적 요소와 인터랙션을 중심으로 설계되었으며,
HTML과 CSS를 기반으로 웹 퍼블리싱 및 레이아웃 구현 역량을 보여주는 것을 목적으로 제작되었습니다.

2. 기획 의도

막연한 목표를 **구체적인 시간 개념(1만 시간)**으로 환산하여 동기 부여

시계·버튼·타이포그래피 등 시각 요소를 활용한 직관적인 사용자 경험(UX) 제공

JavaScript 없이도 구현 가능한 HTML/CSS 중심의 퍼블리싱 역량 검증

3. 주요 기능 및 특징
① 메인 비주얼 구성

중앙에 배치된 시계 이미지로 ‘시간’이라는 핵심 개념을 직관적으로 표현

명확한 시선 흐름을 고려한 레이아웃 설계

② 버튼 및 인터랙션 UI

주요 행동 유도를 위한 버튼 배치

버튼 간 시각적 위계 구분을 통해 사용자의 행동을 자연스럽게 유도

③ 타이포그래피 활용

웹 폰트를 활용하여 메시지 전달력 강화

강조 문구와 일반 문구의 명확한 구분을 통한 가독성 개선

④ 반응형을 고려한 구조

고정 위치(position)와 정렬 속성을 활용하여 화면 크기 변화에도 구조가 무너지지 않도록 설계

4. 사용 기술 스택

HTML5

시맨틱 태그 기반 구조 설계

CSS3

Flexbox 및 Position을 활용한 레이아웃 구현

사용자 인터페이스(UI) 스타일링

GitHub Pages

정적 웹 사이트 배포

5. 본 프로젝트를 통해 보여주고자 한 역량

HTML/CSS 기반 웹 퍼블리싱 기본기

디자이너 관점의 레이아웃 구성 능력

요구사항을 시각적으로 해석하고 구현하는 문제 해결 능력

실제 서비스 형태로 배포까지 진행한 프로젝트 완결 경험

6. 향후 개선 방향

JavaScript를 활용한 실제 시간 계산 기능 추가

사용자 입력값 기반 개인 목표 관리 기능 구현

모바일 환경 최적화를 위한 반응형 고도화

7. 제작자

개인 웹 퍼블리싱 프로젝트

포트폴리오 및 실무 역량 증명을 목적으로 제작
